# 分镜拆分规则

将口播文案拆分为可拍摄的分镜脚本，每个分镜包含：镜头编号、景别、运镜、画面描述、台词、时长。

## 景别术语

- 特写（CU）：突出表情/产品细节
- 近景（MCU）：胸部以上
- 中景（MS）：膝盖以上
- 全景（WS）：全身
- 空镜（ERS）：环境/空镜头

## 运镜术语

- 推：推进感，增强沉浸
- 拉：拉开视野，交代环境
- 摇：左右/上下摇镜头
- 跟：跟随主体移动
- 固定：无运镜，稳当感

## 分镜格式

```
分镜 1
- 景别：特写
- 运镜：固定
- 画面：产品静物，简洁背景
- 台词：就是这个，我用了三个月
- 时长：3秒

分镜 2
- 景别：近景
- 运镜：推
- 画面：人物表情惊讶
- 台词：真的没想到效果这么好
- 时长：4秒
```
